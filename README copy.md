# Create .env file

Copy sample.env and create .env

Edit if required. Install mysql, php, composer etc.

## Versions tested (Others might work)

php8.1
mariadb-server (1:10.5.15-0)
composer (2.4.1)
Laravel-9

# Get started

composer install

composer dumpautoload

php artisan migrate

php artisan db:seed

php artisan serve

Visit http://localhost:8000
